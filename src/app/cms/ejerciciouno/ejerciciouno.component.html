Sergio Ayala ejercicio 1 - Agregando un nav bar lateral

Este proyecto consiste en la creación de un sitio web para un restaurante llamado "Yoko's Kitchen". El sitio web incluye una variedad de páginas con diferentes tipos de contenido, como una página de menú, una página de contacto y una página "Acerca de nosotros". Para mejorar la navegación del sitio web, se agregó un nav bar lateral en todas las páginas.
